# Spots

## Description

Spots is a responsive image-sharing webpage built with HTML and CSS. It includes a profile section, image cards, and responsive layouts for desktop, tablet, and mobile devices. The project follows the provided Figma design and demonstrates responsive web design using HTML and CSS.

## Tech Stack

- HTML5
- CSS3
- Responsive Design
- Flexbox
- CSS Grid
